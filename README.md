# bracelet
a Java data type that implements a Doubly-Linked List structure

To initialize an object of this type you will use the following convention:
```java
Bracelet<T> myBracelet = new Bracelet<T>();
```
where "T" is the object type of your choosing.
For example: 
```java
Bracelet<String> myBracelet = new Bracelet<String>();
```
